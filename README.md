# wbpk_template
使用 webpack5 搭建的一个 vue 项目模板，可以省去大量写 webpack 配置的时间 

## 环境相关配置
- 将 webpack 打包配置进行抽象分离: dev 与 prod
- 在代码中需要注入一些环境变量（例如：BASE_URL、FILE_URL.... ）/injectVar.js
## 资源处理相关
- 文件资源
- css
- js || ts
- vue

## 开发环境相关
- webpack-dev-searver
- 单元测试
- mock 接口集成
- plop 自动生成模块代码
